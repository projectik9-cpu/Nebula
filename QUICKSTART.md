# 🚀 Быстрый старт

## 1. Установка (2 минуты)

```bash
# Установить зависимости
pip install -r requirements.txt

# Создать файл с настройками
copy .env.example .env
```

## 2. Настройка (3 минуты)

### Получить токен бота:
1. Напишите @BotFather в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Скопируйте токен

### Узнать свой ID:
1. Напишите @userinfobot
2. Скопируйте свой ID

### Заполнить .env:
```
BOT_TOKEN=ВАШ_ТОКЕН_ОТ_BOTFATHER
ADMIN_CHAT_ID=ВАШ_TELEGRAM_ID
```

## 3. Запуск (1 секунда)

```bash
python bot.py
```

Готово! 🎉

## 4. Как проверить что работает

### Тест 1: Зайти в бота
1. Найдите своего бота в Telegram
2. Нажмите `/start`
3. Должно появиться главное меню

### Тест 2: Получить реферальную ссылку
1. Нажмите "⭐ Заработать звёзды"
2. Скопируйте ссылку
3. Ссылка вида: `https://t.me/bot_name?start=ВАШ_ID`

### Тест 3: Протестировать реферальную систему
1. Откройте ссылку на другом аккаунте (или попросите друга)
2. Новый пользователь нажимает `/start`
3. Вам должно прийти уведомление
4. В профиле появится +15 ⭐

### Тест 4: Создать промокод
1. В чате с ботом отправьте:
   ```
   /create_promo TEST100 100
   ```
2. Нажмите "🎁 Промокод"
3. Введите: `TEST100`
4. Вам начислится 100 ⭐

### Тест 5: Вывести звёзды
1. Нажмите "💰 Вывести звёзды"
2. Выберите сумму (например, 15 ⭐)
3. Вам (админу) придёт заявка в боте

## 5. Проверка базы данных
