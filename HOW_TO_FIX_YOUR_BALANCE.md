# 🔧 Как исправить ваш баланс

## Ситуация:
- Вчера: пригласили 1 друга → получили 5 звёзд
- Сегодня утром: запустили бота → баланс был 0 (5 звёзд пропали!)
- Сегодня: пригласили 3 друзей → получили 15 звёзд
- Итого: У вас 4 реферала, но вы получили только 15 звёзд вместо 20

## ✅ Быстрое решение (прямо сейчас):

### Вариант 1: Через бота (самый простой)

1. Откройте бота в Telegram
2. Отправьте команду (замените на свой ID):
   ```
   /add_balance 7328504615 5
   ```
3. Готово! Вам добавятся 5 потерянных звёзд

### Вариант 2: Через скрипт

1. Остановите бота (если запущен): `Ctrl+C`
2. Запустите скрипт исправления:
   ```bash
   python fix_balance.py
   ```
3. Следуйте инструкциям на экране
4. Запустите бота снова:
   ```bash
   python bot.py
   ```

## 🛡️ Как предотвратить в будущем:

### 1. Делайте бэкап базы данных КАЖДЫЙ ДЕНЬ

```bash
copy bot.db bot.db.backup
```

Или с датой:
```bash
copy bot.db bot_backup_%date:~-4,4%%date:~-10,2%%date:~-7,2%.db
```

### 2. Проверяйте базу ПЕРЕД запуском бота

```bash
python test_bot.py
```

Это покажет:
- ✅ Всех пользователей
- ✅ Их балансы
- ✅ Количество рефералов

Если видите что баланс 0, хотя были рефералы - **НЕ ЗАПУСКАЙТЕ БОТА!**

Сначала восстановите из бэкапа:
```bash
copy bot.db.backup bot.db
```

### 3. Запускайте бота ВСЕГДА из одной папки

```bash
cd c:\Users\lifes\Downloads\гейбот
python bot.py
```

Если запустите из другой папки, создастся НОВАЯ база данных!

## 💡 Новые админ команды для вас:
