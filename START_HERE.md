## 🏃 Быстрый старт (3 минуты)

### Шаг 1: Установите зависимости
```bash
pip install -r requirements.txt
```

### Шаг 2: Создайте .env файл
```bash
copy .env.example .env
```

### Шаг 3: Заполните .env
1. Получите токен у @BotFather
2. Узнайте свой ID у @userinfobot
3. Впишите в .env:
```
BOT_TOKEN=ваш_токен_от_botfather
ADMIN_CHAT_ID=ваш_telegram_id
```

### Шаг 4: Запустите бота
```bash
python bot.py
```

### Шаг 5: Протестируйте
1. Откройте бота в Telegram
2. Нажмите `/start`
3. Скопируйте реферальную ссылку из "⭐ Заработать звёзды"
4. Откройте ссылку на другом аккаунте
5. Вам придёт уведомление и начислятся звёзды!

## 🎮 Команды бота

### Для пользователей:
- `/start` - Запуск бота (с реферальным параметром или без)
- Кнопка **"⭐ Заработать звёзды"** - Получить реферальную ссылку
- Кнопка **"💰 Вывести звёзды"** - Вывод средств
- Кнопка **"👤 Мой профиль"** - Статистика
- Кнопка **"🎁 Промокод"** - Активация промокода

### Для администратора:
```bash
/create_promo CODE 100   # Создать промокод
/check_user 123456789    # Проверить пользователя
/stats                   # Статистика бота
```

## 🔧 Настройка
