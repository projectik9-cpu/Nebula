## 🔍 TROUBLESHOOTING

### Если бот не отвечает:

**Шаг 1:** Проверьте вебхук
```
https://your-bot.vercel.app/webhook_info
```
Должен показывать URL. Если пустой - установите:
```
https://your-bot.vercel.app/set_webhook
```

**Шаг 2:** Проверьте health
```
https://your-bot.vercel.app/health
```
Должен быть `"healthy"`. Если нет - проверьте логи.

**Шаг 3:** Проверьте логи Vercel
- Deployments → Latest → View Function Logs
- Ищите ошибки

**Шаг 4:** Проверьте Environment Variables
- Settings → Environment Variables
- Все переменные установлены?
- `WEBHOOK_URL` без слеша в конце?

### Если database error:

**Проверьте:**
- [ ] `DATABASE_URL` правильный формат
- [ ] База данных доступна извне
- [ ] Нет ограничений firewall
- [ ] Connection String актуальный

### Если webhook не устанавливается:

**Проверьте:**
- [ ] `BOT_TOKEN` правильный
- [ ] `WEBHOOK_URL` без слеша в конце
- [ ] URL доступен из интернета
- [ ] HTTPS работает (Vercel даёт автоматически)
