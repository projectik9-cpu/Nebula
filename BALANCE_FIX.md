# 🔧 Исправление баланса

## Проблема: Баланс сбросился после перезапуска

### ❓ Почему это могло произойти:

1. **Файл bot.db был удалён** - база данных пересоздалась
2. **Бот запущен из другой папки** - создалась новая база
3. **База данных была повреждена** - редко, но возможно

### ✅ Решение:

## Способ 1: Автоматическое исправление (рекомендуется)

Запустите скрипт для исправления баланса:

```bash
python fix_balance.py
```

Скрипт:
1. Проверит ваш текущий баланс
2. Посчитает сколько должно быть (рефералы - выводы)
3. Предложит исправить если есть расхождения

## Способ 2: Через админ команды в боте

### Вариант A: Установить точный баланс

```
/set_balance <user_id> <сумма>
```

**Пример:**
```
/set_balance 7328504615 5
```

Устанавливает баланс на 5 звёзд (затирает старое значение).

### Вариант B: Добавить к балансу

```
/add_balance <user_id> <сумма>
```

**Пример:**
```
/add_balance 7328504615 5
```

Добавляет 5 звёзд к текущему балансу.

## Способ 3: Вручную через Python

```bash
python -c "from database import Database; db = Database(); db.set_balance(7328504615, 5); print('Готово!')"
```

## 📊 Как посчитать правильный баланс:

### Формула:
```
Правильный баланс = (Количество рефералов × 5) - Сумма выводов
```

### Ваш случай:
- **Рефералов:** 4
- **Заработано:** 4 × 5 = 20 ⭐
- **Выведено:** 30 ⭐ (15 + 15)
- **Должно быть:** 20 - 30 = -10 ⭐ (но баланс не может быть отрицательным)
- **Итого:** 0 ⭐

**Внимание:** Вы вывели больше чем заработали! Это произошло потому что:
1. Вчера награда была 15 звёзд за реферала
2. Вы заработали с одного реферала 15 звёзд (по старой системе)
3. Сегодня награда стала 5 звёзд
4. Новые рефералы дают по 5 звёзд

## 🛡️ Защита от будущих проблем:

### 1. Делайте бэкап базы данных

**Ежедневно:**
```bash
copy bot.db bot.db.backup
```

**С датой:**
```bash
# Windows
copy bot.db bot_%date:~-4,4%%date:~-10,2%%date:~-7,2%.db

# Linux
cp bot.db bot_$(date +%Y%m%d).db
```

### 2. Проверяйте базу перед запуском

Перед запуском бота выполните:
```bash
python test_bot.py
```

Это покажет:
- Всех пользователей и их балансы
- Количество рефералов
- Историю выводов

### 3. Используйте логи

Все начисления и списания записываются в лог. Проверяйте логи если что-то подозрительно.

## 💡 Новые админ команды:

### `/set_balance <user_id> <сумма>`
Устанавливает точный баланс пользователя.

**Когда использовать:**
- Нужно установить конкретное значение
- Исправление ошибок

**Пример:**
```
/set_balance 123456789 50
```

### `/add_balance <user_id> <сумма>`
Добавляет звёзды к текущему балансу.

**Когда использовать:**
- Компенсация за баг
- Бонусы за активность
- Возврат ошибочно списанных звёзд

**Пример:**
```
/add_balance 123456789 5
```

### `/check_user <user_id>`
Показывает полную информацию о пользователе.

**Пример:**
```
/check_user 123456789
```

## ❗ Важно

### База данных НЕ сбрасывается при перезапуске!

Все данные хранятся в файле `bot.db`. Если баланс сбросился, значит:
1. Файл был удалён
2. Бот запущен из другой папки
3. База была повреждена

### Как предотвратить:

1. **Всегда запускайте бота из одной папки**
   ```bash
   cd c:\Users\lifes\Downloads\гейбот
   python bot.py
   ```

2. **Делайте бэкапы**
   ```bash
   copy bot.db bot.db.backup
   ```

3. **Проверяйте перед запуском**
   ```bash
   python test_bot.py
   ```

---

## 🆘 Если проблема повторится:

1. **Сразу остановите бота** (Ctrl+C)
2. **Проверьте базу:** `python test_bot.py`
3. **Восстановите из бэкапа:** `copy bot.db.backup bot.db`
4. **Запустите снова:** `python bot.py`
