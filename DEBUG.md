# 🔍 Инструкция по отладке реферальной системы

## Проблема: Реферальная ссылка не работает

### Шаг 1: Проверьте базу данных
```bash
python test_bot.py
```

Это покажет всех пользователей и их реферов.

### Шаг 2: Проверьте логи бота

Запустите бота и следите за логами:
```bash
python bot.py
```

При переходе по реферальной ссылке вы должны увидеть:
```
INFO:__main__:Получена команда: /start 7328504615 от пользователя 8281043585
INFO:__main__:Обнаружен реферальный параметр: 7328504615
INFO:__main__:Пользователь 8281043585 существует: False
INFO:__main__:Регистрация нового пользователя 8281043585, реферер: 7328504615
INFO:__main__:Реферер 7328504615 существует: True
INFO:__main__:Начисляем 15.0 звёзд рефереру 7328504615
INFO:database:Пользователю 7328504615 начислено 15.0 звёзд
INFO:__main__:Уведомление отправлено рефереру 7328504615
```

### Шаг 3: Проверьте формат ссылки

Правильная ссылка выглядит так:
```
https://t.me/YOUR_BOT_NAME?start=123456789
```

Где `123456789` - это Telegram ID приглашающего.

### Шаг 4: Проверьте, что реферер уже зарегистрирован

Реферер должен **сначала** зайти в бота, нажать `/start`.
Только после этого его реферальная ссылка будет работать.

### Шаг 5: Убедитесь, что друг - новый пользователь

Если друг уже заходил в бота раньше, реферал не засчитается.
Это защита от накрутки.

### Шаг 6: Проверьте, что друг не использует свою ссылку

Нельзя использовать свою реферальную ссылку.

## Типичные ошибки

### Ошибка 1: "Реферер не найден в базе данных"
**Решение:** Реферер должен сначала зайти в бота.

### Ошибка 2: "Пользователь уже существует"
**Решение:** Это нормально. Реферал не засчитывается для старых пользователей.

### Ошибка 3: "Не удалось отправить уведомление"
**Причины:**
- Реферер заблокировал бота
- Реферер удалил диалог с ботом

**Решение:** Звёзды все равно начисляются, просто уведомление не приходит.

## Как протестировать реферальную систему

### Вариант 1: Два аккаунта Telegram

1. На первом аккаунте зайдите в бота и нажмите `/start`
2. Скопируйте свою реферальную ссылку
3. На втором аккаунте перейдите по этой ссылке
4. Проверьте баланс на первом аккаунте

### Вариант 2: С помощью друга

1. Вы заходите в бота первым
2. Отправляете свою ссылку другу
3. Друг переходит по ссылке
4. Вам приходит уведомление и начисляются звёзды

## Проверка через админ-команды

### Проверить конкретного пользователя:
```
/check_user 123456789
```

Покажет:
- Баланс
- Количество рефералов
- Кто пригласил

### Проверить общую статистику:
```
/stats
```

## Частые вопросы

**Q: Почему не начисляются звёзды?**
A: Проверьте логи. Возможно:
- Реферер не зарегистрирован
- Пользователь уже был в боте
- Пользователь перешёл по своей ссылке

**Q: Можно ли начислить звёзды вручную?**
A: Да, используйте промокоды:
```
/create_promo BONUS100 100
```

**Q: Как сбросить базу данных для теста?**
A: Удалите файл `bot.db` и запустите бота заново.

**Q: Звёзды начислились, но уведомление не пришло**
A: Это нормально, если реферер заблокировал бота. Звёзды все равно на балансе.

## Контакты для поддержки

1. Логи бота (в консоли)
2. База данных (через test_bot.py)
3. Админ команды (/check_user, /stats)
