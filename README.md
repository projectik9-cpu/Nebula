# 🌟 Реферальный Telegram Бот

Telegram бот с реферальной системой, проверкой подписки на спонсорские каналы и системой вывода звёзд.

## 🚀 Возможности

- ✅ **Реферальная система** - получайте +5 ⭐ за каждого приглашённого друга
- �️ **Анти-спам защита** - защита от флуда и накрутки
- 🤖 **Защита от ботов** - проверка на реальных пользователей
- 💰 **Система вывода** - вывод от 15 до 350 звёзд (включая Telegram Premium)
- 🎁 **Промокоды** - активация бонусных кодов с улучшенной проверкой
- 👤 **Личный профиль** - статистика и баланс пользователя
- 📊 **Админ панель** - команды для управления ботом
- 📈 **Подробное логирование** - отслеживание всех действий

## 📦 Установка

### 1. Клонирование репозитория
```bash
git clone <url-репозитория>
cd referral-bot
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка конфигурации

Скопируйте `.env.example` в `.env` и заполните данные:
```bash
copy .env.example .env
```

Откройте `.env` и укажите:
- `BOT_TOKEN` - токен бота от @BotFather
- `ADMIN_CHAT_ID` - ваш Telegram ID (для получения заявок на вывод)

**Как узнать свой Telegram ID:**
- Напишите боту @userinfobot
- Или используйте @getmyid_bot

### 4. Настройка награды за реферала (опционально)

Откройте `config.py` и измените размер награды:
```python
REFERRAL_REWARD = 5.0  # Измените на нужную сумму
```

## 🎯 Запуск

```bash
python bot.py
```

## 📱 Использование

### Для пользователей:

1. **Старт бота** - `/start`
2. **Подписка на каналы** - обязательная подписка на все спонсорские каналы
3. **Получение реферальной ссылки** - кнопка "Заработать звёзды"
4. **Вывод средств** - кнопка "Вывести звёзды" (15, 25, 50, 100 или 350 ⭐)
5. **Профиль** - просмотр баланса и статистики
6. **Промокоды** - активация бонусных кодов

### Для администратора:

**Создание промокода:**
```
/create_promo <код> <сумма>
```

Примеры:
```
/create_promo BONUS100 100
/create_promo WELCOME 25
/create_promo VIP500 500
```

**Проверка пользователя:**
```
/check_user <user_id>
```

Пример:
```
/check_user 123456789
```

**Статистика бота:**
```
/stats
```

Показывает:
- Общее количество пользователей
- Общий баланс в обороте
- Количество рефералов
- Статистику выводов

**Управление балансом пользователей:**

Установить точный баланс:
```
/set_balance <user_id> <сумма>
```

Добавить к балансу:
```
/add_balance <user_id> <сумма>
```

Примеры:
```
/set_balance 123456789 50      # Установит баланс на 50 звёзд
/add_balance 123456789 5       # Добавит 5 звёзд к текущему балансу
```

## 🛠️ Технологии

- **Python 3.8+**
- **aiogram 3.4.1** - асинхронный фреймворк для Telegram Bot API
- **SQLite3** - база данных
- **python-dotenv** - управление переменными окружения

## ⚙️ Как работает реферальная система

1. Пользователь получает уникальную ссылку вида `https://t.me/bot_name?start=USER_ID`
2. Когда новый пользователь переходит по ссылке и подписывается на все каналы
3. Пригласивший получает +15.0 ⭐ на баланс
4. Счётчик рефералов увеличивается на 1

## �️ Система безопасности

### Анти-спам защита
- ✅ Ограничение частоты действий (1 секунда между действиями)
- ✅ Защита от флуда в меню и командах
- ✅ Логирование подозрительной активности

- ✅ Валидация ID пользователей

- ✅ Нельзя использовать свою реферальную ссылку
- ✅ Одноразовые промокоды для каждого пользователя
- ✅ Проверка баланса перед выводом

### Подробное логирование
- ✅ Все действия пользователей записываются в лог
- ✅ Отслеживание реферальных переходов
- ✅ Логи начисления и списания звёзд
- ✅ Мониторинг ошибок и подозрительной активности

## 📋 База данных

### Таблица `users`
- `user_id` - Telegram ID пользователя
- `username` - Username пользователя
- `full_name` - Полное имя
- `balance` - Баланс звёзд
- `referrer_id` - ID пригласившего
- `referral_count` - Количество приглашённых друзей
- `created_at` - Дата регистрации

### Таблица `promocodes`
- `code` - Код промокода
- `amount` - Сумма начисления
- `created_at` - Дата создания

### Таблица `used_promocodes`
- `user_id` - ID пользователя
- `code` - Использованный код
- `used_at` - Дата использования

### Таблица `withdrawals`
- `id` - ID заявки
- `user_id` - ID пользователя
- `amount` - Сумма вывода
- `status` - Статус (pending/completed)
- `created_at` - Дата создания
