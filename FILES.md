# 📁 Описание всех файлов проекта

## 🔧 Основные файлы

### bot.py
**Главный файл бота**
- Обработка команд и сообщений
- Реферальная система
- Система вывода
- Промокоды
- Анти-спам защита
- Защита от ботов
- Детальное логирование

**Основные функции:**
- `cmd_start()` - Команда /start с реферальной логикой
- `earn_stars()` - Показ реферальной ссылки
- `withdraw_stars()` - Меню вывода
- `my_profile()` - Профиль пользователя
- `promocode()` - Активация промокодов
- `is_real_user()` - Проверка на бота

**Админ команды:**
- `/create_promo` - Создание промокода
- `/check_user` - Проверка пользователя
- `/stats` - Статистика бота

---

### database.py
**Работа с SQLite базой данных**

**Таблицы:**
- `users` - Пользователи
- `promocodes` - Промокоды
- `used_promocodes` - Использованные промокоды
- `withdrawals` - Заявки на вывод

**Основные методы:**
- `user_exists()` - Проверка пользователя
- `add_user()` - Добавление пользователя
- `get_user()` - Получение данных
- `add_balance()` - Начисление звёзд
- `subtract_balance()` - Списание звёзд
- `increment_referral_count()` - +1 реферал
- `create_promocode()` - Создание промокода
- `activate_promocode()` - Активация промокода
- `create_withdrawal()` - Создание заявки
- `get_statistics()` - Общая статистика

---

### config.py
**Конфигурация бота**

**Параметры:**
- `BOT_TOKEN` - Токен от @BotFather
- `ADMIN_CHAT_ID` - ID администратора
- `REFERRAL_REWARD` - Награда за реферала (по умолчанию 15.0)
- `SPONSOR_CHANNELS` - Список каналов (пока не используется)
